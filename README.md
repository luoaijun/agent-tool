# Agent CLI

**Windows 桌面 AI 任务执行器 · 支持 DeepSeek、小米 MiMo 等多种大模型**

Agent CLI 是一款 Windows 桌面 AI 工具，让你以对话的方式让 AI 自动完成编程、文件管理、搜索等日常任务。AI 可以直接操作你的项目文件、执行命令、搜索网络，像一位坐在你身边的智能助手。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🤖 **多模型支持** | 内置 DeepSeek、小米 MiMo 等模型，可在设置中切换 |
| 📁 **项目隔离** | 每个项目独立上下文，任务不互相干扰 |
| 🔧 **工具调用** | AI 可直接读写文件、执行命令、搜索网络 |
| 📎 **文件拖拽** | 支持拖入图片和文件进行分析 |
| ⚡ **任务队列** | 多任务排队执行，随时暂停/取消 |
| 🔌 **插件系统** | Skill 插件扩展，自定义 AI 能力 |
| 🌐 **跨设备联动** | 支持微信扫码，手机端发起任务 |
| 🎨 **Markdown 渲染** | 对话内容支持 Markdown 实时渲染和代码高亮 |

## 📥 下载安装

### 最新版本：v0.1.1

👉 **[下载 Agent-CLI-Setup-0.1.1.exe](./Agent-CLI-Setup-0.1.1.exe)**

1. 点击上方链接下载安装包
2. 双击运行，选择安装路径
3. 安装完成后桌面会出现 Agent CLI 快捷方式
4. 启动软件，在设置中配置你的 API Key 即可使用

### 系统要求

- Windows 10/11 x64
- 需要配置 AI 模型 API Key（支持 DeepSeek、小米 MiMo 等兼容 OpenAI 接口的模型）

---

## 🚀 快速上手

1. **配置 API Key**：打开设置 → 选择模型提供商 → 填入 API Key
2. **新建项目**：点击左上角项目选择器 → 新建项目
3. **开始对话**：在输入框输入任务描述，AI 会自动执行
4. **拖入文件**：直接把图片或文件拖入输入区，AI 会分析内容

也支持任何兼容 OpenAI API 接口的自定义模型服务。
