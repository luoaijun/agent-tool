核心功能

专为重度 AI 用户设计的桌面任务运行器

🧠

多模型并行

同时运行 2+ 模型，自动路由：推理任务走 Pro，简单任务走 Flash，图像任务走 Vision。

🔀

智能路由

根据任务复杂度自动选择最优模型，推理/视觉/快速模型自动切换，无需手动干预。

📊

Token 追踪

实时显示每次任务的 Token 消耗和费用估算，支持多 Provider 价格配置。

🎙️

本地语音

基于 Windows 本地引擎的 STT 语音输入和 TTS 语音播报，无需联网，隐私安全。

📱

微信授权

任务授权可推送至微信小程序，手机上即可完成审批操作。

⏰

定时调度

预设定时任务，自动执行 AI 工作流，到点自动运行并汇总结果。

📜

时间线视图

任务执行过程以时间线轨道展示，推理、工具调用、子任务一目了然。

🧩

动态插件

支持 Skill/Plugin 动态加载，接入 GitHub 插件、MCP 工具，能力无限扩展。

工作原理

三句话讲清楚

1

输入任务

在对话框中描述你的需求，可以附加图片、文件，选择需要的模型。

→
2

并行执行

Agent CLI 自动分解任务，分配给最合适的模型并行处理，推理 + 工具调用。

→
3

汇总输出

各模型结果汇总展示在时间线中，支持图片、图表、代码高亮等多格式输出。
